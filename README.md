# FlashlightGame

> [!IMPORTANT]
> Running on Unity Editor v6000.3.4f1

## Audio Assets
> [!IMPORTANT]
> Many audio assets are **NOT** included in this repository due to licenses on the audio. The game is able to run without these assets, but if you want to get your hands on theese audio files, follow theese steps.

1. Purchase the assets from the OCs
    - `/Audio/CyberLeaf_UI_SFX`: [https://cyberleaf.itch.io/modern-ui-sfx](https://cyberleaf.itch.io/modern-ui-sfx)
    - `/Audio/LeviVB_UI_SFX`: [https://levivb.itch.io/simple-game-ui-sounds](https://levivb.itch.io/simple-game-ui-sounds)
    - `/Audio/SFX`: [https://sample-kingdom.itch.io/sfx-pack](https://sample-kingdom.itch.io/sfx-pack)
    - `/Audio/Ambiences_SFX`: [https://leohpaz.itch.io/ambiences-perspectives-sfx](https://leohpaz.itch.io/ambiences-perspectives-sfx)
2. Fill the existing folder structure
   1. Extract the zips until you get to the root of the folder.
   2. Drag and drop the root of the folders you downloaded into the appropriate folder in the repository. (There are .gitkeeps in place to guide you)
